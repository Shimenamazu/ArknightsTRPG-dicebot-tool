# ArknightsTRPG-dicebot-tool

CCFOLIAで(非公式)アークナイツTRPGのダイスボットを使用したログにおける成功・失敗などをカウントするツール

利用はこちら<br>
https://shimenamazu.github.io/ArknightsTRPG-dicebot-tool/

PC想定。スマホ？知らん。

【使い方】<br> 1.整形などを行っていないココフォリアのログを読み込む<br> 2.オプションと集計するチャンネルを選択して「集計する」を押す<br>
「表示するキャラクター」は選択したチャンネルでロールを行ったキャラクターをリストアップしています<br>
「表示するキャラクター」のチェックボックスを外すことで非表示にできます<br>
画面右の各項目をクリックすることでソートできます
