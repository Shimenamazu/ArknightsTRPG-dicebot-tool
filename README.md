# ArknightsTRPG-dicebot-tool

CCFOLIAで(非公式)アークナイツTRPGのダイスボットを使用したログにおける成功・失敗などをカウントするツール

利用はこちら
https://shimenamazu.github.io/ArknightsTRPG-dicebot-tool/

PC想定。スマホ？知らん。

【使い方】1.整形などを行っていないココフォリアのログを読み込む2.オプションと集計するチャンネルを選択して「集計する」を押す
「表示するキャラクター」は選択したチャンネルでロールを行ったキャラクターをリストアップしています
「表示するキャラクター」のチェックボックスを外すことで非表示にできます
画面右の各項目をクリックすることでソートできます
